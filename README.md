### 🚀 Настройка и запуск:
<br>

```bash
git clone -b pulse https://github.com/NazarSpokeOfk/My-websites
cd My-websites
npm i gulp
npm i browser-sync
npm i gulp-sass
npm i sass
npm i gulp-clean-css
npm i gulp-rename
npm i gulp-autoprefixer
gulp
```
