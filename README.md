### 📒 Description:
<br>
This website is dedicated to Uber in Russia.
In developing this project, I used not only standard tools but also the Bootstrap library and the SASS preprocessor. The project is primarily built using the Bootstrap grid system. Therefore, to define the layout and dimensions of elements, I used classes that are natively part of Bootstrap.
You can read more about this in the Bootstrap documentation.

### 📒 Description:
<br>
This site is dedicated to heart rate monitors. <br>
This project was built using not only standard tools, but also Gulp, Gulp plugins, and an SCSS preprocessor. The project was primarily built using a custom grid system. 

### 🚀 Setup and Run:
<br>

```bash
git clone -b pulse https://github.com/NazarSpokeOfk/My-websites
cd My-websites
npm i gulp
npm i browser-sync
npm i gulp-sass
npm i sass
npm i gulp-clean-css
npm i gulp-rename
npm i gulp-autoprefixer
gulp
```
